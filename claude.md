# ROMI 프로젝트 개요

## 프로젝트 목적

**ROMI**는 GitHub 이슈를 기반으로 한 팀 내부 지식 베이스 챗봇입니다.

### 핵심 가치

- **과거 작업 검색**: 예전에 개발했던 기능을 자연어로 검색
- **담당자 추적**: 누가 무슨 일을 했는지 쉽게 확인
- **지식 공유**: 팀 내부 문서화 및 지식 베이스 구축

## 기술 방향

- **Backend**: FastAPI (Python)
- **Vector DB**: Qdrant (벡터 검색)
- **AI**: Ollama API (임베딩 + LLM)
- **데이터 소스**: GitHub API (이슈 수집)

## 핵심 기능

1. **이슈 자동 수집**: GitHub Repository 연결 후 이슈 자동 동기화
2. **벡터 기반 검색**: RAG를 활용한 의미 기반 검색
3. **자연어 챗봇**: "로그인 기능을 누가 개발했어?" 같은 질문으로 검색

## 아키텍처 방향

```
사용자 질문 → 벡터 검색 (Qdrant) → RAG 답변 생성 (Ollama) → 관련 이슈 정보 및 레퍼런스 제공
```

## 목표

**"팀의 모든 지식을 한 곳에서 검색하고, 과거 작업을 쉽게 찾아 참고할 수 있는 팀 내부 지식 베이스"**
